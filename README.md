# CSharpConceptsMvcDemo

An ASP.NET Core MVC project that demonstrates:
- Functions vs Methods
- Types of Methods (parameterless, parameterized, void, non-void, instance, static)
- Constructors (default, parameterized, copy, private, static, overloading, chaining)
- Destructor (finalizer)
- `this` keyword
- Parameter passing (`value`, `ref`, `out`)

## Prerequisites
- .NET 6 SDK or later

## Run Locally
```bash
dotnet restore
dotnet run
